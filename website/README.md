# OpenFactstore Website

Static website for OpenFactstore. Open `index.html` in a browser or deploy to GitHub Pages.

## Pages

- `index.html` — Home page: project overview and key features
- `get-started.html` — Getting started guide with Docker Compose quickstart
- `api-docs.html` — API documentation overview and endpoint reference
- `known-issues.html` — Known limitations and production hardening notes

## GitHub Pages Deployment

Push the `website/` directory contents to the `gh-pages` branch, or configure GitHub Pages
to serve from the `/website` folder of the `main` branch.

No build step required — pure HTML, CSS, and JavaScript.
