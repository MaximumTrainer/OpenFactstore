# Factstore CLI

A command-line tool for interacting with the [Factstore](https://github.com/MaximumTrainer/Factstore) Supply Chain Compliance API.

## Installation

### Building from source

Prerequisites: Go 1.21+

```bash
git clone https://github.com/MaximumTrainer/Factstore.git
cd Factstore/cli
make build
# binary is at ./bin/factstore
```

Move the binary to your PATH:

```bash
sudo mv bin/factstore /usr/local/bin/
```

## Configuration

Run the interactive setup wizard:

```bash
factstore configure
```

This saves `~/.factstore.yaml` with your API host and bearer token.

**Alternatively**, use environment variables:

```bash
export FACTSTORE_HOST=https://api.factstore.example.com
export FACTSTORE_TOKEN=my-bearer-token
```

**Or** pass flags per command:

```bash
factstore --host https://api.factstore.example.com --token my-token flows list
```

## Verifying connectivity

```bash
factstore login
```

## Available Commands

### Flows

```bash
factstore flows list
factstore flows get <id>
factstore flows create --name "My Flow" --description "CI pipeline" --attestation-types "SBOM,VULNERABILITY_SCAN"
factstore flows update <id> --name "Updated Name" --attestation-types "SBOM"
factstore flows delete <id>
```

### Trails

```bash
factstore trails list
factstore trails list --flow-id <flow-id>
factstore trails get <id>
factstore trails create \
  --flow-id <flow-id> \
  --commit abc123def456 \
  --branch main \
  --author "Alice Smith" \
  --author-email alice@example.com \
  [--pr-id 42] \
  [--pr-reviewer "Bob"] \
  [--deployment-actor "ci-bot"]
```

### Artifacts

```bash
factstore artifacts list --trail-id <trail-id>
factstore artifacts find --sha256 sha256:abc123...
factstore artifacts create \
  --trail-id <trail-id> \
  --image-name myapp \
  --image-tag v1.2.3 \
  --sha256 sha256:abc123... \
  --reported-by ci-bot \
  [--registry ghcr.io]
```

### Attestations

```bash
factstore attestations list --trail-id <trail-id>
factstore attestations create \
  --trail-id <trail-id> \
  --type SBOM \
  --status PASSED \
  [--details "Generated by syft"]
factstore attestations upload-evidence \
  --trail-id <trail-id> \
  --attestation-id <attestation-id> \
  ./sbom.json
```

### Compliance

```bash
# Assert an artifact meets a flow's requirements
factstore assert --sha256 sha256:abc123... --flow-id <flow-id>

# Full chain of custody report
factstore compliance artifact sha256:abc123...
```

### Webhooks

```bash
factstore webhooks list
factstore webhooks create --source github --secret <secret> --flow-id <flow-id>
factstore webhooks delete <id>
factstore webhooks deliveries <id>
```

## JSON Output

All commands support `--json` to emit raw JSON instead of a table:

```bash
factstore flows list --json | jq '.[].name'
```

## Global Flags

| Flag      | Description                              |
|-----------|------------------------------------------|
| `--host`  | Override configured API host             |
| `--token` | Override configured bearer token         |
| `--json`  | Output results as JSON                   |
